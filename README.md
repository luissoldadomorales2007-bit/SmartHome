# # 🏠 Smart Home Dashboard

Un **dashboard web de control de Smart Home** completamente visual y responsive. Permite ver la **temperatura**, controlar **luces**, monitorear el **consumo energético** y revisar las **cámaras** de manera sencilla, con **modo claro/oscuro** totalmente en CSS, sin JavaScript.

---

## 🔹 Características

- Diseño **moderno y responsive** adaptable a escritorio y móvil.
- Modo **claro/oscuro** usando solo CSS (`checkbox toggle`).
- **Tarjetas individuales** para cada función:  
  - Temperatura actual, mínima y máxima.  
  - Luces con **switch tipo iPhone**.  
  - Consumo energético con barras de progreso.  
  - Cámaras con visualización de imágenes.  
- **Grid flexible**:  
  - Primera fila: Temperatura, Luces y Consumo.  
  - Segunda fila: Cámaras ocupando todo el ancho.
- Todo implementado solo con **HTML y CSS**.
